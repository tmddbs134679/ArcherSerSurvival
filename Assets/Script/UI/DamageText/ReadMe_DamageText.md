# 💥 Damage Text System

플레이어 공격 시 화면에 데미지 숫자를 띄우는 시스템입니다.
 `Object Pooling`을 활용하여 퍼포먼스를 고려한 구조로 설계되어 있으며,
 `DamageText`가 애니메이션을 마치면 자동으로 `DamagePool`로 반환됩니다.
