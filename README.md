## 📆 프로젝트 기간  
**2025.05.08 ~ 2025.05.14** (약 5일)

---


### 📖 개요  
이 프로젝트는 유니티 2D로 만든 프로젝트로,  
- 플레이어(이동, 체력 · 스탯 관리)  
- 몬스터와 전투(AI FSM)
- 스킬 시스템(폭발, 투사체) 
- 스테이지 시스템(Stage 상승에 따른 몬스터 스폰, 무한모드)  
- 아이템 드랍(골드, 포션)  
- 업적 달성
- NPC
등의 핵심 기능을 팀 단위로 구현한 프로젝트입니다.
